F1 by Caio

Aplicação web desenvolvida com React que permite acompanhar informações da temporada da Fórmula 1, incluindo pilotos, equipes, corridas, resultados e classificação do campeonato.

O projeto consome dados de uma API pública open source, exibindo as informações de forma organizada e responsiva.

🌐 Demo

Acesse o projeto online:

👉 https://f1-by-caio.vercel.app/

🏎️ Lista de equipes

📅 Calendário de corridas

📈 Classificação do campeonato (pilotos e construtores)

🔄 Loading e tratamento de erros nas requisições

📱 Interface responsiva

🔌 API utilizada

Os dados são obtidos através da API pública:

Jolpica F1 API

https://api.jolpi.ca/ergast/

Essa API é uma implementação open source baseada no antigo Ergast API, fornecendo dados históricos e atuais da Fórmula 1.

🛠️ Como rodar o projeto

Clone o repositório:

git clone https://github.com/seu-usuario/f1-project.git

Entre na pasta do projeto:

cd f1-project

Instale as dependências:

npm install

Execute o projeto:

npm run dev

📦 Build para produção
npm run build
